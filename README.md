# pristine-styles-docs

This repository hosts the documentation site for [`pristine-styles`](https://github.com/cyril2day/pristine-styles).

It is a small Vue + Vite app that presents the design system through guided sections, reusable UI primitives, and some token examples. The site is meant to be published independently from the core [`pristine-styles`](https://github.com/cyril2day/pristine-styles) package.

## What Lives Here

- The app source in `src/`
- The site entry point in `index.html`
- The Vite and Vitest configuration
- Project tooling and lockfile state

## Getting Started

```bash
pnpm install
pnpm dev
```

## Build And Test

```bash
pnpm build
pnpm test
```
